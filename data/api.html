    Decoupled Blog Demo¶

    This is a simple API that allows our Drupal blog to expose enough data to build a simple decoupled blog.

    This resource lists the available API endpoints.

    Currently these include: users, login_cookie, csrf_token, siteinfo, blog_post, and menus.

    This information is also available at /api/v1.0 which would allow us to version the API in the future, as changes are necessary.
